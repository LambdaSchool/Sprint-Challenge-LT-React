### Questions:

1.  What are PropTypes used for?
# PropTypes are used for checking the values of variables and to make sure they are what you inteded them to be.  Since JavaScript is a dynamically typed language variables can change and so will their types.
1.  What is a lifecycle event in React?
# A lifecycle event are functions to are invoked from before the app is rendered until it's unrendered.  ComponentWillMount, ComponentDidMount, ComponentWillUnmount, etc.
1.  What is a Higher Order Component?
# a higher-order component is a function that takes a component and returns a new component.
1.  What are three different ways to style components in React?
# CSS, Inline Styles, Styledcomponents, or even a preprocessor (less, sass)
