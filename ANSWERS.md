1. PropTypes are used to ensure that the right type of data is being sent to the props.  You can also designate that certain data is required.

2. A lifecycle event is something like ComponentDidMount.  You can alter the state and make changes based on where a component is in it's life cycle.  

3. A higher order component is just like a higher order function.  It's going to take in a Component as an argument, and return a new Component with something added to it.

4. You can style components with inline styles, CSS, and with a library like ReactStrap or bootstrap.
