1.  What are PropTypes used for?
    They are used to check the type of data being passed between components for the assistance of development.

2.  What is a lifecycle event in React?
    A lifecycle event is a waypoint in react where something in a components lifecycle happens such as a component updated, a component mounted, or a component unmounted. 

3.  What is a Higher Order Component?
    A component that is passed another component--its like a higher order function but with components. 


4.  What are three different ways to style components in React?
    1. Inline with the "style" tag.  
    2. With className and .CSS
    3. With component styling.
