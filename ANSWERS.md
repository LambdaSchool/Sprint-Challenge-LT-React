1. PropTypes prevent developers from running into the risk of passing in a wrong data type to a component.

2. Lifecycle event in React is the set of phases in a component's life. Those phases are Birth/Mounting, Growth/Updating, and Death/Unmounting.

3. A Higher Order Component is a function that takes in a component as an argument and returns a new component.
