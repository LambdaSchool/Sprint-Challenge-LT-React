1 What are PropTypes used for?
 
PropTypes are used for validation during development to capture possible issues by validating property types. 

2 What is a life cycle event in React?

Life cycle events are custom methods that get executed at different points in the react life cycle. When a component gets reacted and input into the DOM and when a component updates or gets removed from the DOM. 


3 What is a Higher Order Component?


A technique for reusing a components logic, a HOC is a function that takes in a component as an argument and returns a new component.
